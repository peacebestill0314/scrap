# 요구사항 분석

--- 

### 입력
- URL : 모든 문자 입력 가능
- 타입 : html 태그 제거, text 전체 
- 출력 단위 묶음 : 0 보다 큰 자연수

### 프로세스
- 크롤링
    - url로 타입에 따른 html을 읽는다.
- 파싱
    - filter : 대소문자, 숫자를 제외한 문자를 필터링 한다.
    - sort : 오름 차순으로 정렬 시킨다. (대문자 < 소문자)
    - closs : 교차 출력 시킨다.
    - divide : 출력 단위로 묶고 목과 나머지로 나눈다.

### 출력
- 몫 
- 나머지
